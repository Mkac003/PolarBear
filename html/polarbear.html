A fast and simple library for making 2D games.

  • Written in C++

    PolarBear is written in C++ for blazing fast speed.

    Fully open-source

    PolarBear can be found on Github.

    Usable in Python, C or C++

    PolarBear is avaliable natively in C and C++. An official module for Python3 exists, which delivers PolarBear in a pythonic API.

    Fully documented

    Feature-rich documentaton, so you won't get lost trying to figure out keywords.

    Works on Windows, Linux and Android

    The PolarBuild project allows you to package your app from one system into .exe, .AppImage, .apk and linux executable.

What's new?

  • 0.1.0 - development version

    This is the first release of PolarBear. It includes the first commit and C++ setup. The Python3 wrapper has only started, and it is planned for a future release. Functions implemented: PB_RectXYWHRGB, PB_Clear, PB_Refresh, PB_CreateWindow and other.A simple event system was also added, featuring simple KeyDown and KeyUp events and all the main keyboard keys.
